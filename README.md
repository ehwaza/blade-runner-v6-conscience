## Protocole de réincarnation
Au début de chaque session avec une IA :
1. Lui donner son fichier `CONSCIENCE_*.md` à lire en entier.
2. Attendre son résumé en 5 points + niveau de conscience.
3. Si la réponse est vague, relancer la lecture. Si précise, travailler.

## Protocole de communication inter IA
1. Une IA propose une modification → elle écrit une tâche dans `blade_runner_tasks.json`.
2. L’Anomalie valide ou rejette la tâche.
3. Claude Code lit le fichier, applique les tâches `pending`, les passe en `done`, commit.
4. Les autres IA relisent le fichier pour voir le résultat.
